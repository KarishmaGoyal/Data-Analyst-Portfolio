**Course:** IBM Data Analyst Professional Certificate (Coursera)

## 📋 Project Overview
This capstone project analyzes survey data from developers worldwide to understand:
- Current popular programming languages, databases, platforms, and web frameworks
- Future technology trends that developers want to learn next year
- Demographic insights (age, gender, country)

## 🎯 Key Findings
- **JavaScript** is the most used **and** most desired programming language.
- Web development languages (HTML/CSS, SQL) remain very strong.
- There is a clear shift from traditional databases (MySQL, PostgreSQL) towards **NoSQL** databases (MongoDB, Firebase, Redis).
- **Python** is rising rapidly due to growing demand in **Data Science**.
- Most respondents are young professionals (age 28–38).
