**Project Title:** An Analysis of Current Technology Adoption, Demographics, and Future Trends

**Created by:** Karishma Goyal  
**Date:** April 2026  
**Course:** IBM Data Analyst Professional Certificate (Coursera)

## 📋 Project Overview
This capstone project analyzes survey data from developers worldwide to understand:
- Current popular programming languages, databases, platforms and web frameworks
- Future technology trends developers want to learn
- Demographic insights (age, gender, country)

## 🎯 Key Findings
- **JavaScript** is the most used and most desired programming language
- Shift from traditional databases (MySQL, PostgreSQL) to **NoSQL** (MongoDB, Firebase, Redis)
- **Python** is rising fast due to growing demand in Data Science
- Most respondents are young (28–38 years)
