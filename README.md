**Course:** IBM Data Analyst Professional Certificate (Coursera)

## 📋 Project Overview
This capstone project analyzes real survey data from developers worldwide to understand:
- Current popular programming languages, databases, platforms, and web frameworks
- Future technology trends that developers want to learn next year
- Demographic insights (age, gender, country)

## 🎯 Key Findings
- **JavaScript** is the most used **and** most desired programming language.
- Strong demand for web development skills (HTML/CSS + SQL).
- Clear shift from traditional relational databases (MySQL, PostgreSQL) to **NoSQL** databases (MongoDB, Firebase, Redis).
- **Python** is rising rapidly due to increasing demand in Data Science.
- Most respondents are young professionals (age 28–38 years).
